✈️ Traveloop
🌍 Your Ultimate Intelligent Travel Companion
MongoDB • Express.js • React • Node.js • Tailwind CSS • JWT

A powerful full-stack MERN platform that transforms how users plan, manage, and relive travel experiences — combining itinerary building, smart budgeting, collaboration, and memory preservation into one seamless ecosystem.

📋 Overview
Traveloop is a next-generation travel planning platform built with a vision to:

✅ Simplify complex multi-city trip planning
✅ Provide real-time financial and logistical insights
✅ Enable collaborative travel experiences
✅ Preserve memories through automated storytelling

From planning to post-trip memories, Traveloop handles everything.
